### ЛАБОРАТОРНАЯ РАБОТА

Тема: Использование JSONB, индексов B-Tree и EXPLAIN ANALYZE в PostgreSQL

#### Цель работы

Освоить методы хранения полуструктурированных данных с использованием типа JSONB, научиться создавать и применять индексы B-Tree, а также выполнять измерение времени выполнения запросов с помощью EXPLAIN ANALYZE для оценки эффективности индексирования в PostgreSQL.

Продолжительность выполнения работы: 6 академических часов.

Используемые программные средства

- PostgreSQL 16.x
    
- pgAdmin 8.x
    
- Операционная система Windows 10/11, Ubuntu 22.04+ или macOS 13+
    
- Query Tool, входящий в состав pgAdmin
    
#### Требования к программному обеспечению

На компьютере должны быть установлены PostgreSQL 16 и pgAdmin 8. Пользователь должен обладать правами на создание баз данных, таблиц и индексов. Для выполнения работы рекомендуется наличие не менее 2 ГБ оперативной памяти и 500 МБ свободного дискового пространства.

#### Краткие теоретические сведения

Тип данных JSONB предназначен для хранения документов в формате JSON в бинарном виде. В отличие от текстового хранения JSON, формат JSONB обеспечивает более эффективный поиск и обработку данных.

Индекс B-Tree является наиболее распространенным типом индексов PostgreSQL и используется для ускорения поиска, сортировки и сравнения значений.

Команда EXPLAIN ANALYZE позволяет получить план выполнения запроса и фактическое время его выполнения. Данный инструмент используется для оценки эффективности запросов и сравнения различных способов организации данных.
